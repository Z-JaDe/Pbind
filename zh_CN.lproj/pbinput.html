PBInput

PBInput 继承至 UITextField，并扩展了以下属性：

type

输入类型，不同的类型将会为其创建不同的辅助输入视图：

类型 辅助输入控件 备注
text 文本键盘
password 文本键盘 显示为 *
number 数字键盘 0-9
decimal 数字键盘 0-9 与 '.'
phone 电话号码键盘 0-9 与 '*', '+', '-', ';'
url 文本键盘
email 文本键盘 包括 '@' 字符
date UIDatePicker 年月日，3列
time UIDatePicker 时分秒，3列
datetime UIDatePicker 年月日时分秒，6列
month UIDatePicker 年月，2列
select PBOptionPicker 选项值，1列

name

输入控件域，用于构造 <name, value> 键值对。

value

输入控件值，用于构造 <name, value> 键值对。

text

用于显示控件值对应的文本。

min and max

用于限制输入的最小值与最大值，根据不同的输入类型接收不同的值：

类型 最小值/最大值 示例
number 整型 20
decimal 浮点型 20.0
date 由 '-' 连接的年月日 2015-2-1
month 由 '-' 连接的年月 2015-2

unit

输入单位字符串，如果指定了将为输入控件添加一个设置 text 为 unit 的 UILabel 作为 rightView。

unitColor

输入单位文本颜色。仅当指定 unit 时生效。

options

用于 select 类型的输入控件，必填。格式为 NSArray<NSDictionary<NSString *, id> *>, 形如：

[{text:'a', value:0}, {text:'b', value:1}]

其中的 text 与 value 是保留字段，text 用于显示，value 作为选项的值。

selector

用于 select 类型的输入控件，可选。设置的值为 辅助输入视图 的类名（必须是 PBInputView 的子类）。

format

用于格式化输入文本，对于不同的输入类型：

类型 支持的格式化 e.g.
number 字符串格式化 "%lld piece(s)", 20 => "20 piece(s)"
decimal 字符串格式化 "%.1lf", 20 => "20.0"
任何 分段格式化 "## ## ##", 123456 => "12 34 56"

placeholderColor

未输入时的默认文本颜色。

selectedTextColor

选中态的输入文本颜色。

hidesCursor

是否隐藏光标，默认否。

maxlength

可输入的最大文本长度。长度由 [NSString length] 来决定。

maxchars

可输入的最大字节数。字符计为1，unicode计为2。通过 [NSString charaterLength] (Pbind自定义分类) 来计算。

pattern

用于验证每个输入的字符，使用正则。

示例：配置 '[0-9]' 时，如果输入 'a'，将被忽略。

validators

由字典构成的一个数组，用于校验整个输入文本。每个字典需要包含两个字段：pattern 与 tips。 pattern为一个正则，用于匹配输入文本，tips为匹配失败时的错误提示。

示例：配置 [{pattern: '^S\w?$', tips: 'Should starts with S'}] 时，如果第一个输入的不是 'S'，将提示 'Should starts with S'。

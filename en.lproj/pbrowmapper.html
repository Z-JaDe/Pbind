Row mapper

The base controls of Pbind are PBTableView, PBCollectionView and PBScrollView, each of them use row(linear) layout.

Hereby we take row as base element by which we can also achieve the reusability of view.

The names could be configured are:

clazz

The class name for each row, default is UITableViewCell.

nib

The xib file name for each row, default as clazz.

style

The style(UITableViewCellStyle) for each cell. If configured, the clazz will take PBTableViewCell as default to support it.

layout

The layout file name by what to create the subviews for the view in the row.

id

The resuable identifier for the cell(item) of the row. Default is "clazz @layout style". Supports for PBTableView and PBScrollView`.

height

The height for the row. Default is -1 (auto-height). Supports for PBTableView and PBScrollView`.

estimatedHeight

The estimated height for the row. Default is -1 (do not estimated). Supports PBTableView. If the height is unset, default to 44.

size

The size for the item(UICollectionViewCell) in the row. Supports for PBCollectionView.

floating (Incubating)

The floating for the row, supports for PBScrollView and usually for the submit button which needs to stick in the bottom.

Accept values:

  • (1) PBRowFloatingTop stick in top for vertical scroll view
  • (2) PBRowFloatingLeft stick in left for horizontal scroll view
  • (3) PBRowFloatingBottom stick in bottom for vertical scroll view
  • (4) PBRowFloatingRight stick in right for horizontal scroll view

actions

The actions for the row, includes:

name refer to delegate note
willSelect PBActionMapper willSelectCellAtIndexPath: 将选中行事件
select PBActionMapper didSelectCellAtIndexPath: 已选中行事件
willDeselect PBActionMapper willDeselectCellAtIndexPath: 将取消选中行事件
deselect PBActionMapper didDeselectCellAtIndexPath: 已取消选中行事件
delete PBRowActionMapper editActionsForRowAtIndexPath: 左滑删除按钮事件
edits PBRowActionMapper editActionsForRowAtIndexPath: 左滑自定义按钮组事件

Thereinto, the PBRowActionMapper is based on PBActionMapper and extend following names to configure the appearance of the button displayed by swipe-to-left.

name class note
style UITableViewRowActionStyle the button style
title NSString the title of the button
backgroundColor UIColor the background color of the button

style 支持：

style 按钮样式 效果
:normalStyle UITableViewRowActionStyleNormal 灰色背景
:deleteStyle UITableViewRowActionStyleDestructive 红色背景

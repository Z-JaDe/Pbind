行配置

Pbind的核心控件为 PBTableView, PBCollectionView, PBScrollView，均基于行布局。

故以 行 为基础元素，来完成视图可复用。

可配置的属性包括：

clazz

用于定义每一行的视图类，默认为 UITableViewCell。

nib

用于定义视图对应的 xib 文件名，默认与 clazz 同名。

style

用于定义 UITableViewCellStyle，如果有指定值，clazz 将默认为 PBTableViewCell，以支持该属性。

layout

用于定义子布局，其创建的子View将以AutoLayout的形式添加到 clazz 所构建的父View中。 这个配置也为实现每个 行单元 的复杂布局提供了可能性。

id

用于定义Cell的可复用标识，默认为："clazz @layout style"。

height

用于定义行高，适用于 PBTableView 与 PBScrollView。默认为 -1，自动行高。

estimatedHeight

用于定义预估行高，用于自动行高的 PBTableView。默认为 44。

size

用于定义列表项(UICollectionViewCell)大小，用于 PBCollectionView。

floating

用于定义行悬浮，适用于 PBScrollView，通常用作固定于底部的提交按钮等等。

支持的值为：

  • (1) PBRowFloatingTop 顶部悬浮，适用于纵向滚动布局
  • (2) PBRowFloatingLeft 左悬浮，适用于横向滚动布局
  • (3) PBRowFloatingBottom 底部悬浮，适用于纵向滚动布局
  • (4) PBRowFloatingRight 右悬浮，适用于横向滚动布局

actions

每一行的交互行为，包括：

行为 映射类 代理方法 描述
willSelect PBActionMapper willSelectCellAtIndexPath: 将选中行事件
select PBActionMapper didSelectCellAtIndexPath: 已选中行事件
willDeselect PBActionMapper willDeselectCellAtIndexPath: 将取消选中行事件
deselect PBActionMapper didDeselectCellAtIndexPath: 已取消选中行事件
delete PBRowActionMapper editActionsForRowAtIndexPath: 左滑删除按钮事件
edits PBRowActionMapper editActionsForRowAtIndexPath: 左滑自定义按钮组事件

其中 PBRowActionMapper 基于 PBActionMapper，扩展以下属性来支持左滑出现的按钮外观配置：

属性 类 描述
style UITableViewRowActionStyle 按钮样式
title NSString 按钮标题
backgroundColor UIColor 按钮背景色

style 支持：

style 按钮样式 效果
:normalStyle UITableViewRowActionStyleNormal 灰色背景
:deleteStyle UITableViewRowActionStyleDestructive 红色背景

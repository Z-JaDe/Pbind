导航配置

在根视图 plist 的 Root 节点下支持通过 nav 来配置导航栏，其下定义的属性将会映射到当前 Controller 的 navigationItem：

属性 类型 对应 含义
title String title 导航栏标题
left Dictionary leftBarButtonItem 左导航按钮
lefts Array leftBarButtonItems 左导航按钮数组
right Dictionary rightBarButtonItem 右导航按钮
rights Array rightsBarButtonItems 右导航按钮数组

导航按钮配置

导航按钮通过字典来定义，最终转换为 UIBarButtonItem。

参数包括两部分，首先是初始化参数，不同的初始化参数决定了使用不同的系统 API 来创建按钮。

初始化参数 API 效果
type initWithBarButtonSystemItem:target:action: 系统内置样式
image
& style		 initWithImage:style::target:action: 自定义背景图按钮
title
& style		 initWithTitle:style::target:action: 自定义标题按钮

其中 type 支持：

type 系统图标类型 效果
:done UIBarButtonSystemItemDone
:cancel UIBarButtonSystemItemCancel
:edit UIBarButtonSystemItemEdit
:save UIBarButtonSystemItemSave
:add UIBarButtonSystemItemAdd
:compose UIBarButtonSystemItemCompose
:reply UIBarButtonSystemItemReply
:share UIBarButtonSystemItemAction
:organize UIBarButtonSystemItemOrganize
:bookmarks UIBarButtonSystemItemBookmarks
:search UIBarButtonSystemItemSearch
:refresh UIBarButtonSystemItemRefresh
:stop UIBarButtonSystemItemStop
:camera UIBarButtonSystemItemCamera
:trash UIBarButtonSystemItemTrash
:play UIBarButtonSystemItemPlay
:pause UIBarButtonSystemItemPause
:rewind UIBarButtonSystemItemRewind
:fastforward UIBarButtonSystemItemFastForward
:undo UIBarButtonSystemItemUndo
:redo UIBarButtonSystemItemRedo
:pagecurl UIBarButtonSystemItemPageCurl

image 配置的是图片名称，将从 [Pbind allResourceBundles] 中读取。

style 支持：

style 边框样式 效果
:plainStyle UIBarButtonItemStylePlain 无
:doneStyle UIBarButtonItemStyleDone 字体加粗

此外，另一部分参数控制已创建按钮的属性：

控制参数 描述
hidden 是否隐藏，默认 NO
enabled 是否可点击，默认 YES
action 按钮点击事件，参考PBAction组件

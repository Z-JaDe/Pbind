Navigation mapper

We support configure the navigation bar by define a nav property under the Root node in Plist.

All the properties defined in nav would be map to the current controller's navigationItem:

name type map to note
title String title navigation title
left Dictionary leftBarButtonItem the left bar button item for the navigation bar
lefts Array leftBarButtonItems the left bar button items for the navigation bar
right Dictionary rightBarButtonItem the right bar button item for the navigation bar
rights Array rightsBarButtonItems the right bar button items for the navigation bar

BarButtonItem

The item configured by a Dictionay and turns to be a UIBarButtonItem finally.

The dictionary contains two part of parameters, first part is for initializing by what we select the related API to create a UIBarButtonItem.

name API result
type initWithBarButtonSystemItem:target:action: a system built-in item
image
& style		 initWithImage:style::target:action: a custom background image item
title
& style		 initWithTitle:style::target:action: a custom title item

Thereinfo we support types:

type UIBarButtonSystemItem effect
:done UIBarButtonSystemItemDone
:cancel UIBarButtonSystemItemCancel
:edit UIBarButtonSystemItemEdit
:save UIBarButtonSystemItemSave
:add UIBarButtonSystemItemAdd
:compose UIBarButtonSystemItemCompose
:reply UIBarButtonSystemItemReply
:share UIBarButtonSystemItemAction
:organize UIBarButtonSystemItemOrganize
:bookmarks UIBarButtonSystemItemBookmarks
:search UIBarButtonSystemItemSearch
:refresh UIBarButtonSystemItemRefresh
:stop UIBarButtonSystemItemStop
:camera UIBarButtonSystemItemCamera
:trash UIBarButtonSystemItemTrash
:play UIBarButtonSystemItemPlay
:pause UIBarButtonSystemItemPause
:rewind UIBarButtonSystemItemRewind
:fastforward UIBarButtonSystemItemFastForward
:undo UIBarButtonSystemItemUndo
:redo UIBarButtonSystemItemRedo
:pagecurl UIBarButtonSystemItemPageCurl

image refer to the background image name, we will find the image from [Pbind allResourceBundles].

style supports:

style UIBarButtonItemStyle effect
:plainStyle UIBarButtonItemStylePlain nothing
:doneStyle UIBarButtonItemStyleDone bold font for title

The other part of parameters are used to configure the properties of the created item:

name note
hidden Whether hide the item, default is NO
enabled Whether the item can be clicked, default is YES
action The click action for the item, refer to Action mapper

PBButton

PBButton extends from UIButton and add some properties for ease of configurating.

title

The title for the normal state, and the other states can be configured by:

name state
highlightedTitle UIControlStateHighlighted
selectedTitle UIControlStateSelected
disabledTitle UIControlStateDisabled

image

The image name for the normal state, and the other states can be configured by:

name state
highlightedImage UIControlStateHighlighted
selectedImage UIControlStateSelected
disabledImage UIControlStateDisabled

backgroundImage

The background image name for the normal state, and the other states can be configured by:

name state
highlightedBackgroundImage UIControlStateHighlighted
selectedBackgroundImage UIControlStateSelected
disabledBackgroundImage UIControlStateDisabled

backgroundColor

The background color for the normal state, and the other states can be configured by:

name state
highlightedBackgroundColor UIControlStateHighlighted
selectedBackgroundColor UIControlStateSelected
disabledBackgroundColor UIControlStateDisabled

If the highlightedBackgroundColor was not set, default to the background color with alpha component 0.8.

If the disabledBackgroundColor was not set, default to the background color with alpha component 0.2.

If the selectedBackgroundColor was not set, default to the background color.

action

The action triggered by click the button. Refer to Action component.
